# Tricity Tech Company + HR Contact Database

A research database of technology companies and HR/recruiter contacts across India's **Tricity** — **Chandigarh, Mohali, Panchkula & Zirakpur** — aimed at fresher / internship / 0–1 yr **Python & AI-ML developer** roles.

## Files

| File | What it is |
|---|---|
| [`chandigarh_mohali_tech_hr_database.md`](chandigarh_mohali_tech_hr_database.md) | Human-readable database — company rankings, priority outreach shortlist, LinkedIn/cold-email templates, and a "go tomorrow / apply this week" action list. |
| [`chandigarh_mohali_tech_hr_database.csv`](chandigarh_mohali_tech_hr_database.csv) | Machine-readable dataset (Excel-ready), 18 columns per company. |
| [`chandigarh_mohali_tech_hr_dossier.html`](chandigarh_mohali_tech_hr_dossier.html) | Searchable, filterable browser dossier — open it directly in any browser. |

## Ground rules

- **No fabricated data.** No invented emails, phone numbers, names, or LinkedIn URLs. "Not found publicly" means no public source existed — it is never a placeholder guess.
- Every named contact has a real, search-verified LinkedIn profile plus at least one piece of hiring-related evidence.
- Sourced via live public web search: LinkedIn, company career pages, Naukri / Indeed / Glassdoor / Internshala, and business directories.

> Contact details listed here were gathered from public sources for personal job-search outreach. Please use them respectfully and in line with the platforms' terms.
